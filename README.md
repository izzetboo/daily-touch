# daily-touch

Minimal daily mindfulness app built with Expo + React Native.

## What it does

- Single screen with one centered **Hold** button.
- While holding, vibration pulses become stronger over time.
- On release, vibration stops and a glassy card appears with:
  - a compliment
  - a short interesting fact
  - a small mindful task

All content is local placeholder data (no backend, no APIs).

## Run locally

```bash
npm install
npm start
```

Then open on iOS/Android simulator or the Expo Go app.
